# 古蓬顺风车系统

本地顺风车预订系统，支持古蓬到柳州、来宾、南宁等线路。

## 功能特点

- 🚗 **多角色系统**：乘客、司机、管理员三种角色
- 📍 **地图定位**：支持在线地图和离线地图
- 🎫 **班次管理**：发布、预订、管理班次线路
- 💰 **价格计算**：自动计算行程价格
- 📱 **移动适配**：完美适配手机端
- 💾 **本地存储**：数据存储在浏览器本地
- 📷 **地点图片**：司机可上传目的地参考图片
- 📲 **PWA支持**：可添加到手机桌面使用

## 部署方法

### Vercel 部署（推荐）

1. 注册 [Vercel](https://vercel.com) 账号
2. 点击 "New Project"
3. 导入本项目文件
4. 自动部署完成

### GitHub Pages 部署

1. 创建 GitHub 仓库
2. 上传所有文件
3. 开启 GitHub Pages 功能
4. 访问生成的链接

## 使用说明

1. **乘客**：
   - 查看可用班次
   - 预订座位
   - 查看个人订单

2. **司机**：
   - 发布班次
   - 查看乘客列表
   - 上传地点图片
   - 管理订单状态

3. **管理员**：
   - 管理司机账户
   - 上传收款码
   - 下载离线地图
   - 数据备份恢复

## 技术栈

- HTML5 / CSS3 / JavaScript
- Leaflet 地图库
- Font Awesome 图标
- LocalStorage 本地存储
- PWA 渐进式网页应用

## 注意事项

1. 数据存储在浏览器本地，清除浏览器数据会丢失
2. 建议定期使用"备份数据"功能导出数据
3. 首次使用建议下载离线地图
4. 微信中打开可能功能受限，建议在浏览器中打开
